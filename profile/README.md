# Wallbit Workflows

Open-source tooling ecosystem for programmable financial workflows on top of the Wallbit API.

The goal of this ecosystem is to make financial operations more programmable through workflows, automation, reusable tooling and developer-first infrastructure.

## Projects

### wallbit-cli
CLI for interacting with the Wallbit API, running workflows and automating financial operations.

### wallbit-registry
Public registry for publishing, versioning and discovering reusable wallbit-cli YAML workflows (pull/publish via the CLI).

### wallbit-go
Go SDK for building integrations, services and workflow tooling on top of the Wallbit API.

### wallbit-workflow-builder
Workflow builder and skill system for creating reusable YAML workflows compatible with `wallbit-cli`.
